1 # CTex
2 
3 Generate `LaTeX` math equations from `C` source code.
4 
5 `CTex` is the tool to simplify documentation generation for source code with a lot of math expressions.
6 
7 `CTex` can help you in the following ways:
8 
9 * Quickly generates `LaTeX` equations for huge amount of math expressions in a source code.
10 * Helps with representation of math formula in `LaTeX` format if you don't familiar with syntax.
11 * Generates `Doxygen`-friendly output
12 
13 `CTex` is developed under macOS, but is set-up to be highly portable. As a result, it runs on Windows and on a variety of Unix flavors as well.
14 
15 ## Installation
16 
17 ```
18 git clone https://github.com/galarius/ctex.git
19 cd ctex
20 mkdir build && cd build
21 cmake -G "Unix Makefiles" ..
22 make && make install
23 ```
24 
25 ## Usage
26 
27 * `ctex.exe <input.c> <output.c>`
28 
29 * `ctex.exe -i` - interactive mode
