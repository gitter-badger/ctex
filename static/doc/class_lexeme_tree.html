Detailed Description

Lexeme Tree.

Lexeme tree build algorithm: I) Partial tree contains only lexemes considered to be transform operators, like functions, operations (see LexemeLibrary::is_toperator)... II) Tree extension by using only lexeme positions relatively to transform operators in tree
